Setup

Up and running in 2 minutes

1

Install the Addon

Download the addon ZIP. In Blender: Edit > Preferences > Add-ons > Install.

2

Start the Server

Open the N panel in 3D Viewport. Go to the BM Pro tab. Click "Start Server".

3

Configure MCP Client

Add the MCP server path to your Claude Desktop, Cursor, or Windsurf config.

4

Start Creating

Ask your AI to create objects, set up lighting, build materials — all in natural language.

See It In Action

Professional studio shot in 8 seconds

Ask in plain English. Blender MCP Pro handles the rest.

"Set up a professional studio shot of the cube. Three-point lighting, brushed metal with subtle edge reflections, render at 1920x1080."
create_three_point_lighting("Cube")
→ Key light (Sun, energy=5), Fill light (Area, energy=2), Rim light (Spot, energy=3)

set_principled_bsdf("Material", metallic=0.9, roughness=0.3, base_color=[0.7, 0.7, 0.8])
→ Brushed metal material applied

add_modifier("Cube", "BEVEL", segments=3, width=0.02)
→ Edge bevel for subtle reflections

render_image("./studio.png", 1920, 1080, 128)
→ Rendered to studio.png (2.3s)

FAQ

Frequently asked questions

What is MCP (Model Context Protocol)? ▼
MCP is an open protocol by Anthropic that allows AI assistants like Claude to use external tools. Blender MCP Pro registers 120+ tools that let the AI directly control Blender — creating objects, setting up materials, managing lights, animating keyframes, and more.
How is this different from the free blender-mcp? ▼
The free ahujasid/blender-mcp covers basic object and material operations, but lacks light control, modifiers, animation, shader node trees, and advanced geometry nodes. It also has known bugs (Issues #189, #190) and connection instability. Blender MCP Pro fixes all these issues and adds 80+ exclusive tools.
Which Blender versions are supported? ▼
Blender MCP Pro supports Blender 3.6 LTS and later, including Blender 4.x. It's tested against every major release.
Can I use this with Cursor or Windsurf? ▼
Yes. Blender MCP Pro works with any application that supports the MCP protocol. This includes Claude Desktop, Cursor, Windsurf, and any other MCP-compatible client.
What happens after the free trial? ▼

Does it work offline? ▼
License verification happens at startup, but the system caches your license status for up to 72 hours. After initial verification, you can work offline without interruption.
